### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/shionart/dnsmultiprotest.git
   ```
2. Install MVN Dependencies and skip tests
   ```sh
   mvn install -DskipTests
   ```
3. run apps
   ```sh
   mvn spring-boot:run
   ```
